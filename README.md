# 🚀 22 MW CFBC Hybrid Dynamic Digital Twin

> AI-Driven Hybrid Dynamic Digital Twin for a 22 MW CFBC Power Plant using Physics-Based Modeling, Machine Learning, Streamlit, SQLite and Power BI.

---

## 📌 Project Overview

This project presents an industry-oriented Hybrid Dynamic Digital Twin framework for a 22 MW CFBC Power Plant. The system integrates:

- Physics Twin (MATLAB Simulink)
- AI Twin (Random Forest + XGBoost)
- Energy Management System (EMS)
- SQLite Historian
- Streamlit Dashboard
- Power BI Analytics
- Predictive Maintenance
- Plant Health Monitoring

The framework combines first-principles modeling with data-driven AI techniques to enable intelligent monitoring, performance analysis, and predictive maintenance of thermal power plants.
